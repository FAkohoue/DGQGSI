# DGQGSI: Desired-Gain Selection Index and Desired-Gain QGSI Tools

Package providing tools for:

- desired-gain selection index optimization,

- desired-gain adapted QGSI computation from trait GEBVs,

- automatic construction of quadratic desired-gain matrices,

- comparison of DG and QGSI outputs,

- a high-level wrapper for full pipeline execution.
